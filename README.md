# Web Scraping and Automate Forms

## Running Application

Open your terminal in the project directory and run:


`cd scrapers` or `cd automate-forms`

```
bundle install
```

### To run scrapers

`ruby any_ruby_script_file.rb`

### To run automated forms

`rspec any_rspec_script_file.rb`
